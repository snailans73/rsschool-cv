## Code Sample
Codewars (hhttps://www.codewars.com/kata/515de9ae9dcfc28eb6000001/train/javascript) "Split Strings":

```js
function solution(str){
   var arr = str.split('')
   var res = []

   for(let i = 0; i<arr.length; i+=2){
     if(arr[i+1]) res.push(arr[i]+arr[i+1])
     else res.push(arr[i]+'_')
   }

  return res
}
```
